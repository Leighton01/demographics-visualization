## Canada's Population Landscape

Using data from the Human Mortality Database and Statistics Canada, this project analyses contemporary demographic patterns in Canada, including mortality trends by age over time, with a focus on immigration policy effects. Data visualizations highlight key patterns and support interpretive insights.

The final video walkthrough of the analysis is available here: [link](https://drive.google.com/file/d/1wECi3SO0qqH190NYpeItQ16Nyvn7B1ll/view?usp=sharing)
